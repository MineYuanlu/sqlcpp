Detailed Description

参数占位符映射关系管理

记录 SQL 语句中参数('?')的顺序与代码中参数的顺序的映射关系

允许用户通过 bind 将参数顺序转换为SQL语句所需的顺序

bind 需要一个能够使用<<运算符传入数据的 binder, 并将参数顺序绑定到 binder 中

例如:

static auto select = Select("name").from("table").where(Field("age") > VAR(1) && Field("time") < VAR(0));
static auto sql = select.build();
static auto var = select.get_var_map();
CustomBinder binder;
int time = 100;
int age = 18;
std::string ignore;
var.bind(binder, time, age, ignore);
