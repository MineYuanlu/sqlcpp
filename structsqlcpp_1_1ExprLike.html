Detailed Description

基本表达式

OpExpr, CaseExpr, FuncExpr 被算作基本表达式, 使用 ExprLike 来包装。

而 ExprLike, FieldLike, ValueLike, Select 等不被算作基本表达式, 使用 Expr 来包装
